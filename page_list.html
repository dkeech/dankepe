Page List

  • Order

    Create an order. Uses SELECT queries to populate dropdown lists, and an INSERT query to put the new data into a table. Items are listed below and can be edited in the field or deleted from the order using the buttons to the right, which use ALTER and DELETE queries.

  • Products

    Add a product. Uses SELECT queries to populate dropdown lists, and an INSERT query to load data into a table. Uses a SELECT query to populate a table showing available products.

  • Brands

    Uses an INSERT query to put data into a table, and a SELECT query to show available brands on the page.

  • Customers

    Uses an INSERT query to send data to a table, and ALTER and DELETE queries to modify tables.

  • Orders

    Uses a SELECT query to show existing orders, and a DELETE query to delete one.
